# Dog_app

A web application for browsing dog images randomly using [Dog.CEO's](https://dog.ceo/) DOG API 

## Live Demo 

- [Dog_app](https://asylra.github.io/Dog_app/)

## Local Installation

- Clone the project repo

```
$ git clone https://github.com/AsylRA/Dog_app.git
$ cd Dog_app
$ npm install
$ http://127.0.0.1:3000
```

## Tools

- JavaScript
- HTML, CSS
